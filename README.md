# Travel-Planning-App
Travel Planning & Budgeting App

Effortlessly plan, organize, and manage your group trips with our mobile app! Designed with Flutter and powered by Firebase, this app lets users create detailed travel plans, schedule activities, manage companions, track budgets, and collaborate in real time.

Features:

Travel Plan Creation: Plan trips with title, date range, destination, and notes.

Detailed Itinerary: Add scheduled activities (with location, date/time, cost, members assigned).

Budget & Expense Tracker: Record trip expenses, set budget estimates, and auto-calculate totals. Compare planned vs. actual spending.

Group Collaboration: Add members, assign tasks, and allow everyone to update plans together using real-time sync with Firebase.

Tasks & Responsibilities: Assign tasks to each companion and track completion status.

PDF Export (“Resibo”): Generate and export travel details and expenses for sharing or documentation.

Authentication: Secure login with Firebase Auth; supports multi-user collaboration.

Unique Features: Visualize expenses by category (chart), upload photos for activities, and enjoy real-time updates.

Tech Stack:

Frontend: Flutter (Dart)

Backend/Database: Firebase (Firestore/Realtime Database), Firebase Auth

Other: PDF generation, optional Firebase Storage & Messaging

Project Objectives:

Build a mobile app with full CRUD functions for travel plans, itinerary, expenses, members, and tasks.

Provide a collaborative experience for group trip planning.

Ensure user-friendly UI/UX with clear validations and error handling.

Deliver at least one unique feature (e.g., expense charts, PDF export).
